# OptiSched Requirements

## Objective

Develop a backend platform capable of generating optimized university schedules using mathematical optimization techniques.

## Problem Statement

Creating university timetables manually is time-consuming and prone to conflicts involving professors, classrooms and course schedules.

OptiSched aims to automate this process while respecting scheduling constraints.

## Target Users

- University administrators
- Department coordinators
- Academic staff

## MVP Features

- Professor management
- Subject management
- Classroom management
- Availability management
- Semester management
- Schedule generation
- Schedule visualization

## Future Features

- Authentication
- Multi-campus support
- CSV import/export
- Optimization reports
- Multiple optimization algorithms
